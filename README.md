# JCTask

 Android {JSON} Placeholder API Task
#### Tools and Technologies 
1- Koin
2- Room DB with Live Data
3- Retrofit 
4- Kotlin Coroutines
5- RxJava
6- Work Manager



#### How To Run
1- import with Android Studio https://github.com/nayelfares/JCTask.git
2- Download and Sync Gradle and Click on Run.
